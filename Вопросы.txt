Endpoint - указывает на точку подключения, предоставляют клиентам доступ к функциональным возможностям службы
Binding - по какому формату будет проходить обмен, указывает сетевые протоколы, механизмы кодирования и транспортный уровень
Contract - описание каждого метода к которому открыт доступ
Host - адрес локального узла на котором распологается служба
Behavior - Каждый элемент behavior определяется атрибутом name и обеспечивает либо предоставленное системой поведение (например <throttling>), либо пользовательское поведение. 
Если имя не задано, то элемент behavior будет соответствовать поведению по умолчанию для службы или конечной точки.

Сессии бывают:-
security sessions
reliable sessions
TCP sessions
